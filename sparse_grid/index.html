Sparse Grid Finite Difference Codes¶

This is a set of MATLAB codes that automates sparse grids so that practitioners can use the sparse grids without having to know the implementation details. This set of codes were originally written to test using sparse grids in solving Hamilton-Jacobi-Bellman equations. Adapting any toolbox can require a large up-front time-cost, so you should skim over Tutorials to see if this toolbox is relevant for your application. From my previous search of toolboxes on sparse grids, you might want to check out:

  • SG++
  • Tasmanian

that have more/other features, but do not have finite difference operators implemented. [1]

If you are coming from the economics literature on solving economic models with heterogeneous agents or solving Hamilton-Jacobi-Bellman equations (using finite difference methods), you should also consider my other toolbox

  • Mean-Field Games Toolbox NOT PUBLIC YET

to which this toolbox makes references.
