Sparse Grid Finite Difference Codes¶

This is a set of matlab codes that automates sparse grids so that practitioners can use the sparse grids without having to know the implementational detail. This set of codes were originally written to test using sparse grids in solving Hamilton-Jacobi-Bellman equations. So far even things are only optimized if they are a bottleneck in the entire process of solving HJB equations. People who are more interested in a general application of the sparse grids might want to check out:

  • SG++
  • Tasmanian

Unfortunately, neither of them contain implementation of finite difference operators, but extending them should be straightforward following [ahn2018sparse]. For low-dimensional problems, there should not be much difference in speed between this implementation and the C++ implementation given above.

For people who are only interested in the mean-field games application or solving Hamilton-Jacobi-Bellman equations (with finite difference method), you can consider my other toolbox

  • MFG solver NOT PUBLIC YET

to which this toolbox makes references.
