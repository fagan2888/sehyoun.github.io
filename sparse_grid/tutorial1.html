Simple Tutorial¶

The implementation given here is intended so that users can use sparse grid approximations without the understanding of the underlying mathematical/implementational theory. For people who do not know anything about sparse grids, reading section 3 of [ahn2018sparse] is sufficient.

Function Approximation¶

As an example of function approximation, we will approximate \(f(x,y) = \sin(x) + \cos(y)\) on \([0,1]^2\) with sparse grids. To start, we need to add the folder to the toolbox in path, and initialize a sparse_grid object by

addpath('/path/to/folder/src');
grid = sparse_grid;

Since the dimensionality of the problem is 2, we set

grid.n_dim = 2;

Given this, we can build up the sparse grid of level 5 by calling

grid.init_grid(5);

Through this function call, the class takes care of everything under the hood. Hence, the evaluation operators and differentiation operators are set. First, let’s make a plot of the \(f(x,y)\). The position of the grid points are saved in x_i in matrix with columns corresponding to dimensions and rows grid points. Hence, we can get the values of the function at the corresponding grid points via

f_val = sin(grid.x_i(:,1)) + cos(grid.x_i(:,2));

We can visualize via scatter plot

figure(1);
scatter3(grid.x_i(:,1), grid.x_i(:,2), f_val, '.');

We can evaluate the forward finite difference operator by using d_forward. Since it is a cell-array with differential operators, we can evaluate the forward difference operator in x-dimension and compare to the true value of \(cos(x)\)

figure(2);
forward_derivatives = grid.d_forward{1}*f_val;
scatter3(grid.x_i(:,1), grid.x_i(:,2), forward_derivatives, '.');
hold on;
scatter3(grid.x_i(:,1), grid.x_i(:,2), cos(grid.x_i(:,1));
